Учебный проект, выполненный по урокам.

Описание:
Web-приложение с возможностью регистрации, которое позволяет отправлять текстовые сообщения и изображения с тегами.
Представляет из себя Spring Boot-приложение, с подключённой базой данных PostgreSQL. Отображение в браузере выполнено с использованием шаблонизатора FreeMarker,
для управления правами доступа использовался Spring Security.

Функционал:
- отправка текстовых сообщений и изображений с указанием тега;
- поиск сообщений по тексту сообщения или по тегу;
- регистрация;
- роли (user, administrator);
- редактирование прав доступа пользователей через администратора.
